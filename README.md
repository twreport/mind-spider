# MindSpider

AI-powered sentiment crawler for Chinese social media platforms.

## Supported Platforms

- Xiaohongshu (小红书)
- Douyin (抖音)
- Kuaishou (快手)
- Bilibili (哔哩哔哩)
- Weibo (微博)
- Tieba (贴吧)
- Zhihu (知乎)

## Installation

```bash
pip install -e .
playwright install chromium
```

## Configuration

Copy `.env.example` to `.env` and fill in your settings.

## Usage

```bash
python main.py
```
